# bamazon

bamazon is a shopping store system built with MySQL, node.js. In bamazon user could place the order by item id and the system will instant replay if store have enough quantity to process the order.

## How to use bamazon

1. Run the bamazonCustomer.js with node.
2. Following the prompt questions
   - Select item by enter the item id.
   - Enter quantity desired.
3. System will check if there are enough stock quantity for the order.
4. Thank you for shopping at bamazon.

## NPM Package installed for this app:

1. Mysql (https://www.npmjs.com/package/mysql#performing-queries)
2. Inquirer (https://www.npmjs.com/package/inquirer)
